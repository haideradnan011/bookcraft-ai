# 📚 BookCraft AI - محرر الكتب الذكي

أداة احترافية لتحويل النصوص الجاهزة إلى كتب منسقة وجميلة تلقائياً.

## 🎯 الرؤية

نقل تجربة الكتابة والنشر من العملية اليدوية المحبطة إلى تطبيق ذكي يفهم الكاتب ويساعده على إنتاج كتب احترافية بسهولة.

## ✨ المميزات الأساسية

### 1. تنظيم النصوص الذكي
- تقسيم تلقائي للفصول
- إنشاء عناوين وعناوين فرعية
- ترتيب الهوامش والحواشي
- إنشاء فهرس تلقائي

### 2. ترقيم الصفحات الذكي
- فهم حجم النص وطول الفقرات
- توازن الصفحات
- احترام بدايات ونهايات الفصول
- ترقيم تلقائي احترافي

### 3. تنسيق متعدد الأنماط
- **فلسفي**: مساحة تنفس كبيرة، خطوط أنيقة
- **أدبي**: تدفق قراءة سلس، تنسيق درامي
- **أكاديمي**: هيكل منظم، ترقيم دقيق
- **روائي**: سرد ممتع، تنسيق سردي
- **تأملي**: صمت بصري، خطوط دقيقة
- **Minimal Clean**: بساطة وأناقة
